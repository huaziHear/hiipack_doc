# hiipack代理使用方法

## 推荐方式

要使用 hii 的代理功能，两个步骤即可：

1. **创建代理配置文件**
2. **启动本地服务时添加参数: **`--proxy --open`** 或者 **`-xo`

> 提示:
> 
> 1. `-x`或者`--proxy` ==&gt; 启动代理服务
> 
> 2. `-o`或者`--open` ==&gt; 打开浏览器窗口，默认Chrome

## 手动配置

如果不让 hiipack 打开浏览器窗口，代理服务启动后，需要自己手动设置浏览器代理或者系统代理。

代理地址：

```

127.0.0.1:4936

```

这样浏览器里面所有的请求就会经过 hiipack 的启动在`4936`的服务。
